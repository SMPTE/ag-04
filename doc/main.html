Templates

HTML Templates

The following Document types shall be created using the HTML Tooling workflow as defined in and use the HTML template repo at :

  • Standard (ST)
  • Recommended Practice (RP)
  • Engineering Guideline (EG)
  • Administrative Guideline (AG)
  • Operation Manual (OM)

This does not currently apply to other Documents types: Registered Disclosure Document (RDD), Advisory Note (AN), Engineering Report (ER), Overview Document (OV). Support in the HTML tooling for those Document type may come at a later date.
